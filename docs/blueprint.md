# **App Name**: NutriGenius AI

## Core Features:

- Streamlined User Onboarding: User-friendly onboarding form to collect health, dietary, and preference information optimized for mobile devices.
- AI Diet Plan Generation: AI-powered analysis of user input to create personalized diet plans, leveraging an LLM 'tool' for determining optimal nutritional guidelines based on user preferences, geographic location, and basic health factors.
- Diet Plan Display: Display of 7-day rotating meal plans with options and alternatives optimized for mobile display, along with portion size recommendations and shopping list features.
- Progress Input: Simple interface for daily progress tracking (weight, energy levels, meal completion) using straightforward inputs, minimizing the need for external device integrations.
- Progress Dashboard: Dashboard visualization of progress against goals, showing trends and milestones with charts and graphs in a mobile-friendly layout.
- Recipe Curation Tool: Admin tool to create a curated and regularly updated recipe library.
- Expert Query Routing: Accept dietary queries based on location and health status, and forward the query to an expert.
- Dietary Plan Generation: Use an LLM tool to filter results based on multiple experts and health factors, without using an external database.
- Personalized Plan Display: Display the custom diet plan based on the response of the selected expert in the related geographic location.
- Offline Plan Download: Allow the option to download the diet plan for offline access.
- Preference Setting: Option to specify food allergies and health preferences.
- Knowledge Addition: Add knowledge to experts by connecting to a Google Drive folder and adding text files.
- Expert Creation: Option of creating an expert by feeding them knowledge.
- Expert Selection: When a question is asked, decide which expert(s) should respond.
- Daily Progress Tracking: Allow users to see their diet plan and enter details daily to monitor progress.
- Progress Analytics: Display progress analytics based on user input.
- Personalized Tips: Provide tips to improve analytics based on user's daily inputs, using an LLM tool.
- User Authentication: Each user should have their own login credentials
- Payment-Based Access Control: Admin option to stop users from using the app based on their payment status.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4), reminiscent of health and trustworthiness.
- Background color: Light, desaturated blue (#E3F2FD) creates a calm, clean feel.
- Accent color: Analogous cyan (#00BCD4), with brightness and saturation clearly distinct from the primary.
- A simple, intuitive grid layout to ensure ease of navigation and content discovery, optimized for mobile devices.
- Body and headline font: 'Alegreya', a serif font known for readability. Optimized for mobile viewing.
- Use clean, line-style icons to represent food groups and dietary options.  Icons should be easily recognizable on smaller screens.
- Subtle transitions to provide visual feedback without disrupting the user experience. Animations should be efficient to conserve battery life on mobile devices.
